# HITA课表

[App下载](https://github.com/bdbd2yy/HITA_L/releases/download/2.0/app-release.apk)

暂不支持IOS端，但是你可以 vibe 自己的HITA！

## 项目背景
项目来自哈尔滨工业大学（深圳）2018级本科生大一年度立项。原名HITSZ助手，重构版改名为HITA课表
HITA_Lite 在此基础上进一步进行了结构重构与功能精简，移除了冗余模块，保留核心课表能力，愿景是成为一个小而美的版本。

## 应用简介
HITSZ助手是面向哈尔滨工业大学（深圳）学生的工具类APP+校园在线APP。
- ###### UI设计
	极简主义，支持主题色更换，支持深色模式
- ###### 课表及日程
	课程表、日程表
- ###### 教务服务
	导入课表、选课、成绩、空教师查询
